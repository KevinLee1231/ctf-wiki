# Gone Missing 1

## 题目简述

题目给出一个可旋转的街景全景，要求在地图上标出失踪的火属性宝可梦所在位置。网页只在猜测坐标落入目标附近时返回 flag，因此决定性任务是从建筑与城市空间布局识别地标。

官方服务配置为该题保存了目标坐标：

```text
59.9168622, 10.7297468
```

## 解题过程

全景中的主体是正对长轴景观的宫殿式建筑。将建筑正面、两翼比例和前方开放空间裁出后做反向图片搜索，可以匹配到挪威奥斯陆王宫
（The Royal Palace, Oslo）。再用地图或卫星图核对宫殿、Slottsparken 公园和 Karl Johans gate 城市轴线，排除外观相似的欧洲宫殿。

把地图标记放到奥斯陆王宫附近即可通过距离校验。仓库中的坐标
`59.9168622, 10.7297468` 与该地标一致，也为识别结果提供了官方侧复核。

当年的玩家复盘还记录了“先对宫殿截图做反向图片搜索、直接得到 Oslo Royal Palace”的现场路径；相关关键信息已完整转写在上文，原记录可作为来源查看：[UMD CTF 2023 赛后复盘](https://dree.blog/blog/umd-ctf-2023/)。

服务返回：

```text
UMDCTF{I_b3t_rainbolt_c0uld_g3t_th1s_!n_thr33_s3c0nd5}
```

## 方法总结

- 核心技巧：优先裁剪辨识度最高的地标建筑做反向图片搜索，再用周边道路与公园布局二次验证。
- 识别信号：完整宫殿立面和明确城市轴线通常比路牌 OCR 更稳定，适合先走 landmark matching。
- 复用要点：反向搜索只能给候选；最终应使用地图空间关系和坐标一致性完成交叉验证。
