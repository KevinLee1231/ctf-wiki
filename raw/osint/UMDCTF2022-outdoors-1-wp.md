# UMDCTF2022 Outdoors 1 Writeup

## 题目简述

题目称 Jason Heyson 开始直播“环境、户外或现实生活”相关内容，要求找到直播。该题依赖 2022 年比赛期间在线的 Twitch 账号和实时观众信息；仓库官方解题说明明确标注它现在可能已经无法按原路径复现。

## 解题过程

题面中的 “live-streaming” 和 “live” 指向直播平台，目标平台是 Twitch。姓名也是用户名构造提示：

```text
Heyson -> Hey's son -> son_of_hey
```

原赛时访问 Twitch 用户 `son_of_hey` 可以找到 Jason 的户外直播。直播的观众列表或互动区域中有一个看似随机的十六进制字符串；把它按字节从 hex 解码为 ASCII，即可得到：

```text
554d444354467b796565747d
```

解码：

```python
bytes.fromhex("554d444354467b796565747d").decode()
```

结果为：

```text
UMDCTF{yeet}
```

当前 Twitch 直播及当时的在线观众状态没有被仓库保存，所以无法再独立观察该十六进制账号或消息；用户名、平台、编码机制和最终值均来自仓库中的官方 `writeup.txt`。

## 方法总结

实时 OSINT 题的证据寿命很短。比赛时应立即保存目标账号 URL、直播截图、观众列表和时间戳，否则赛后只能依靠复盘。本题的两处主要推理分别是用姓名双关得到 `son_of_hey`，以及把随机外观的观众标识按十六进制解码；不能在直播消失后声称过程仍可完整复现。
