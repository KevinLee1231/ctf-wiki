# Gone Missing 3

## 题目简述

题目给出一段乡间道路全景，要求定位失踪的电属性宝可梦。画面中没有可直接提交的地址，关键线索来自电线杆样式、农田景观、路边中文海报、道路朝向和电力线走向。

官方服务配置中的目标坐标为：

```text
23.6444599, 120.5169568
```

## 解题过程

先从环境判断国家或地区。画面中的全高黑黄相间条纹电线杆在台湾乡间较常见，周围水田、低矮建筑和道路形态也与台湾西部平原相符。继续观察路边海报，可从中文地名中识别出“云林”，于是把搜索范围缩小到台湾云林县。

仅确定县级区域仍不足以提交。接下来利用全景视角中的罗盘确定道路朝向，再在地图和街景中比对以下特征：

1. 道路的弯折方向和邻近岔路位置；
2. 电线杆位于道路哪一侧，以及电线跨越道路的角度；
3. 两侧农田、建筑和树木的相对位置；
4. 周围地势是否保持台湾西部平原的开阔特征。

逐段排除后可定位到云林县境内的目标道路。官方坐标
`23.6444599, 120.5169568` 与道路方向和电力线布局吻合。

公开赛后记录补充了条纹电线杆、海报中的“云林”、罗盘、道路方向及电力线是主要定位证据；上文已转写完整推理链，来源见：[UMD CTF 2023 赛后复盘](https://dree.blog/blog/umd-ctf-2023/)。

最终得到：

```text
UMDCTF{c0mpass3s_and_p0wer_l!n3s_FTW!}
```

## 方法总结

- 核心技巧：先用地区特有的基础设施缩小国家或地区，再用可读地名和空间几何完成精确定位。
- 识别信号：黑黄条纹电线杆与中文只能给出候选区域，真正用于落点的是罗盘方向、道路弯折和电力线布局。
- 复用要点：街景题应同时记录观察方向；只比较“看起来相似”的景物，容易在同一乡镇的重复道路中误判。
