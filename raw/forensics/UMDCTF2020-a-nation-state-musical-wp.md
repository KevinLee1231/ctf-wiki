# A Nation State Musical

## 题目简述

附件是一次可疑网络通信的 PCAP。攻击者刻意使用乌克兰来源地址制造归因假象，但数据包负载中包含反向 shell 连接目标和一段自然语言文本；需要从通信内容而不是表面源地址判断实际国家。

## 解题过程

在 Wireshark 中按 TCP 流追踪，或用 Scapy 输出较长的原始负载。流中能看到反向 shell 尝试连接：

```text
37.46.96.0
```

同一段负载还包含哈萨克语文本。对该目标地址做地理归属核对，可确认其指向哈萨克斯坦。相比之下，数据包头部的乌克兰源地址只是生成脚本设置的诱饵，不能作为攻击者来源的决定性证据。

最终答案为：

```text
UMDCTF-{Kazakhstan}
```

## 方法总结

网络取证不能只依据源 IP 做归因。源地址可能被伪造或作为刻意误导，而应用层负载、回连目的地、语言和基础设施归属可以相互印证；本题应把多条独立证据关联后再下结论。
