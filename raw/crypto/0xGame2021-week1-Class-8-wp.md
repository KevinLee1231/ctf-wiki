# week1Class 8

## 题目简述

题图把 16 个大写字母分别编码成八类古典密码，并说明同一对括号内的符号属于同一种密码。需要逐项识别后按从左到右、从上到下的顺序拼接。

## 解题过程

题图中各位置对应的密码类型与解码结果如下：

| 位置 | 密码类型 | 结果 |
| --- | --- | --- |
| 1、6 | 盲文 | `C`、`L` |
| 2、7 | 跳舞的小人 | `L`、`N` |
| 3 | 猪圈密码 | `A` |
| 4、11、16 | 手机九键 | `S`、`L`、`B` |
| 5、9、14 | 摩尔斯电码 | `S`、`D`、`T` |
| 8、12 | 银河字母 | `R`、`D` |
| 10、15 | 培根密码 | `D`、`N` |
| 13 | 键盘围形 | `V` |

第 13 位由键盘上的 `c`、`f`、`g`、`b` 围出 `V`。第 15 位的培根密码存在字母表版本差异，题目仓库同时接受 `N` 与 `O` 两种结果。因此可提交：

```text
0xGame{CLASSLNRDDLDVTNB}
```

或：

```text
0xGame{CLASSLNRDDLDVTOB}
```

## 方法总结

混合古典密码题应先利用括号、排版和重复符号完成分组，再逐组套用对应字母表。培根密码等存在多套历史字母表时，应检查题目判定范围，不能把版本差异误当作唯一解。
