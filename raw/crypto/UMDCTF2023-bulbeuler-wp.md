# Bulbeuler

## 题目简述

生成器把 flag 转成二进制。对每个比特随机选取 $x\in\mathbb F_p$，再输出一对有限域元素

$$
(a,b)=(g^x,g^y).
$$

若当前位为 1，令 $y=p-1-x$；若当前位为 0，则独立随机选择 $y$。附件给出素数 $p$ 和所有 $(a,b)$，目标是逐位恢复 flag。

## 解题过程

有限域乘法群 $\mathbb F_p^\*$ 的阶为 $p-1$，因此

$$
g^{p-1}=1.
$$

当编码位为 1 时，

$$
ab=g^xg^{p-1-x}=g^{p-1}=1\pmod p.
$$

当编码位为 0 时，$x+y$ 是随机值，$ab=1$ 只会以约 $1/(p-1)$ 的概率偶然发生；在 256 位素数下可忽略。

所以逐行检查两数乘积即可判位：

```python
bits = ""
for a, b in pairs:
    bits += "1" if a * b % p == 1 else "0"

flag = int(bits, 2).to_bytes(len(bits) // 8, "big")
print(flag.decode())
```

输出文件保持了原始 flag 的逐字节、最高位在前顺序，无需反转位序。得到：

```text
UMDCTF{I_d0nt_th1nk_bulby_!s_th3_n3xt_3ul3r_tbh}
```

## 方法总结

- 核心技巧：把两个有限域幂相乘，利用指数和在群阶 $p-1$ 上归零来识别编码位。
- 识别信号：按位输出成对的 $g^x,g^y$，且某一分支令两个指数之和等于 $p-1$，通常意味着可以通过乘积或逆元关系直接判别。
- 复用要点：无需离散对数。先寻找群运算下的不变量，往往比恢复随机指数简单得多。
