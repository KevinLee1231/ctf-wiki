# oneline6ryp7o

## 题目简述

Flag 主体固定为 67 个字符，每个字符只能是 `6` 或 `7`。题目把完整 flag 按大端字节序解释为整数，并要求它能被

```python
~(6 + ~7) ** 67
```

整除。由于字符 `6` 与 `7` 的 ASCII 值只差 1，每个位置的选择可转化为一个二进制位。

## 解题过程

先化简模数。Python 中 `~7=-8`，所以括号内为 $-2$；幂运算后再执行外层按位取反：

$$
M=\mathop{\sim}(-2)^{67}
 =\mathop{\sim}(-2^{67})
 =2^{67}-1.
$$

因此有 $2^{67}\equiv1\pmod M$。

把 67 个位置全部设为 `6`，记完整基准 flag 对应的整数为 $B$，需要补上的余数为：

```python
M = ~(6 + ~7) ** 67
a = -int.from_bytes(b'SEKAI{' + b'6' * 67 + b'}') % M
```

将主体第 $i$ 个字符从 `6` 改成 `7`，会给大端整数增加

$$
2^{8(67-i)}
\equiv 2^{-8i\bmod67}\pmod M.
$$

因为 $\gcd(8,67)=1$，当 $i=0,\ldots,66$ 时，$-8i\bmod67$ 恰好遍历全部 67 个比特位置。也就是说，每个主体字符唯一控制余数 `a` 的一位，且不同字符不会落到同一位。读取这些位即可决定选 `6` 还是 `7`：

```python
body = ''.join('67'[a >> (-i * 8 % 67) & 1] for i in range(67))
print(f"SEKAI{{{body}}}")
```

将输出重新转成整数并检查 `% M == 0`，即可验证推导和字符顺序均正确。

输出为：

```text
SEKAI{6777676667666666677676776776777766777777777776777767777776677666666}
```

## 方法总结

固定字母表且字符值相邻时，大整数同余常可线性化为比特选择问题。本题还利用了 $2^{67}-1$ 的循环移位性质：字节权重的指数可模 67 化简，而 8 在模 67 下可逆，因此 67 个字符正好对应 67 个独立比特，比枚举 $2^{67}$ 个字符串有效得多。
