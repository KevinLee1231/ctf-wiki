# 0xGame2048

## 题目简述

附件只包含一串由多种 Unicode 文字区段字符组成的文本：

```text
Жఱ൲ඌיય೬ࢶЖۍךะtঋළม۹ρԊҽඹ
```

题目名中的 `2048` 和这种高码位字符分布共同指向 Base2048。该编码使用最多 2048 个符号承载数据，主体字符相当于每个表示 11 bit，因此输出通常由跨多个文字区段的 Unicode 字符组成。

## 解题过程

将附件文本交给支持 Base2048 的解码器，选择 UTF-8 文本输出。这里不是逐字符替换或加密，解码器会按照 Base2048 字符表把符号还原为位流，再按字节解释为明文。

解码结果为：

```text
0xGame{W3lc0me_t0_0xG4me!!!}
```

## 方法总结

- 核心技巧：根据题名提示和 Unicode 字符分布识别 Base2048 编码。
- 识别信号：短文本由多个语言区段的可打印字符组成，同时题面出现 `2048`、`base` 或“高密度编码”等提示。
- 复用要点：先保留原始 Unicode 文本，避免复制过程发生规范化或字符替换；解码结果应再检查是否形成合法 flag 格式。
