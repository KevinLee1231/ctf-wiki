# week1Factor

## 题目简述

题目给出 textbook RSA 的模数 $n$、公钥指数 $e=65537$ 和密文 $c$。模数只有约 128 位，可以直接分解；题目描述“不会有人从 1 跑到根号 $n$”也在提示不要使用朴素试除。

## 解题过程

已知参数为：

```text
n = 177726843226591634556244030635816071333
e = 65537
c = 49549088434190402681586345733724247189
```

使用成熟的整数分解算法或公开因数数据库可得到：

```text
p = 9735957770491659841
q = 18254685097880877413
```

先检查 $pq=n$，再计算 $\varphi(n)=(p-1)(q-1)$、$d\equiv e^{-1}\pmod{\varphi(n)}$，最后求 $m\equiv c^d\pmod n$：

~~~python
from Crypto.Util.number import *

c = 49549088434190402681586345733724247189
p = 9735957770491659841
q = 18254685097880877413
n = p * q
phi = (p - 1) * (q - 1)
e = 0x10001
d = inverse(e, phi)
m = pow(c, d, n)
print(long_to_bytes(m))
~~~

输出为：

```text
b'we1come_t0_rs4'
```

## 方法总结

- 核心方法：先分解较小的 RSA 模数，再按标准私钥恢复流程解密。
- 识别特征：模数位数明显不足、没有填充机制，且题面主动暗示整数分解。
- 注意事项：不要从 1 到 $\sqrt n$ 朴素试除；使用外部因数数据库只是获取 $p,q$ 的一种方式，正文必须保留参数、公式和乘积校验，避免解法依赖外链。
