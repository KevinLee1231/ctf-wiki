# LLL-I

## 题目简述

flag 被平均切成四段并转为整数，作为矩阵 $M$ 的第一行；其余三行是小于 $2^{90}$ 的随机噪声。程序再左乘一个由单位上、下三角矩阵相乘得到的整数矩阵 $C$，只公布 $CM$。目标是利用格基不变性和 LLL 恢复 $M$ 中的短向量。

## 解题过程

构造 $C$ 的两个三角矩阵对角线均为 1，因此：

$$
\det(C)=1
$$

也就是说 $C$ 是整数上的幺模矩阵，$CM$ 的行向量与 $M$ 的行向量生成同一个格。公开矩阵的行虽然是很大的整数线性组合，但对其做 LLL 规约，可以重新找到原格中的短向量。

完整 SageMath 脚本如下：

```python
from Crypto.Util.number import long_to_bytes

B = Matrix(ZZ, [
    [
        1849784703482951012865152264025674575,
        2664848085955925754350117767673627932,
        2099783527396520151610274180590854166,
        1020558595577301617108111920545804527,
    ],
    [
        1207449566811121614020334020195802372,
        1954621976999112878661150903673543232,
        1326050406731534201574943690688237338,
        1361813208094227445768111591959011963,
    ],
    [
        888810907577479776819993141014777624,
        1216302736807928240875874427765340645,
        1027359437421599069599327712873719567,
        238961447144792739830554790892164336,
    ],
    [
        60622164517940943037274386912282,
        82958508138755168576836012717468,
        70072118066826856564329627650828,
        16296740862142507745322242235326,
    ],
])

reduced = B.LLL()

for row in reduced.rows():
    candidate = b"".join(
        long_to_bytes(abs(Integer(value)))
        for value in row
    )
    if candidate.startswith(b"0xGame{"):
        print(candidate.decode())
        break
```

规约后的第一条短向量恰好是 flag 四段整数的相反数；取绝对值并逐段转回字节，得到：

```text
0xGame{04679c42-2bc1-42b2-b836-1b0ca542f36b}
```

## 方法总结

左乘幺模矩阵只更换格基，不改变格本身。题目试图用大系数线性组合隐藏原矩阵，但原始行仍是格中的异常短向量，LLL 可以将其找回。识别这类题时，应优先检查变换矩阵的行列式，以及秘密向量与噪声向量相对公开组合的长度差。
