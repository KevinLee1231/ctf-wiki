# New_Python!

## 题目简述

题目要求恢复管理员密码。普通用户注册后可以下载 `guess.py`，其中给出一组泄露了 CRT 参数 $d_p=d\bmod(p-1)$ 的 RSA 数据，并提示管理员密码由 `uuid.uuid8(a, b, c)` 生成。三个参数分别藏在 RSA 明文、HTTP 响应头和 `/auth` 接口中。

管理员登录成功后可以把输入直接交给 `os.popen()` 执行，因此得到 UUID 密码后提交 `env` 即可读取环境变量中的 flag。源码里的 Flask `secret_key = "0xGame{RSA1_UUID8}"` 只是会话签名密钥，不是最终 flag。

## 解题过程

### 由 $d_p$ 泄露恢复参数 $a$

附件给出 $n$、$e$、密文 $c$ 和 $d_p$。因为 $ed_p\equiv1\pmod{p-1}$，存在整数 $k$ 满足：

$$
ed_p-1=k(p-1),\qquad 1\le k<e
$$

枚举 $k$，当 $k\mid(ed_p-1)$ 时计算候选 $p=(ed_p-1)/k+1$，再用 $p\mid n$ 验证即可恢复因子。下面的脚本只使用 Python 内置大整数运算：

```python

t = e * dp - 1

for k in range(1, e):
    if t % k != 0:
        continue
    p = t // k + 1
    if n % p != 0:
        continue

    q = n // p
    phi = (p - 1) * (q - 1)
    d = pow(e, -1, phi)
    m = pow(ciphertext, d, n)
    plaintext = m.to_bytes((m.bit_length() + 7) // 8, "big")
    print(plaintext.decode())
    break
```

输出为：

```text
key{rsaisfunbutisitweborcrypto}
```

取花括号中的字节串并按大端序转成整数：

```python
key = b"rsaisfunbutisitweborcrypto"
a = int.from_bytes(key, "big")
print(a)
```

得到：

```text
183915192278352122275137263475187826728085592578452428749304943
```

### 获取 $b$、$c$ 并生成 UUIDv8

应用给所有响应添加如下响应头，因此抓取任意页面即可得到 $b$：

```text
X-Frame-Options: b = 120604030108
```

目录枚举可发现 `/auth`，其 JSON 响应直接给出 $c$：

```json
{"c":"7430469441","token":"Token is Useless, But You Can Catch This Page!"}
```

所以 $b=120604030108$、$c=7430469441$。`uuid.uuid8(a, b, c)` 是 CPython 3.14 新增的接口，三个自定义字段分别保留低 48、12、62 位；超出字段宽度的高位会被截断。在 Python 3.14 中可直接生成密码：

```python
import uuid

a = int.from_bytes(b"rsaisfunbutisitweborcrypto", "big")
b = 120604030108
c = 7430469441
print(uuid.uuid8(a=a, b=b, c=c))
```

若没有 Python 3.14，也可以按 UUIDv8 的字段布局手工组装，并写入版本位 `8` 和 RFC 4122 变体位 `10`：

```python
import uuid

a = int.from_bytes(b"rsaisfunbutisitweborcrypto", "big")
b = 120604030108
c = 7430469441

value = (
    ((a & ((1 << 48) - 1)) << 80)
    | (8 << 76)
    | ((b & ((1 << 12) - 1)) << 64)
    | (0b10 << 62)
    | (c & ((1 << 62) - 1))
)
print(uuid.UUID(int=value))
```

两种方式都得到管理员密码：

```text
63727970-746f-849c-8000-0001bae3f741
```

使用用户名 `admin` 和该密码登录，在管理员命令框提交 `env`。`app.py` 会执行 `os.popen(cmd).read()`，返回结果中包含：

```text
FLAG=0xGame{Only_Python14&15_UUID8}
```

最终 flag：

```text
0xGame{Only_Python14&15_UUID8}
```

## 方法总结

本题把 RSA CRT 参数泄露、HTTP 信息搜集和 UUIDv8 字段布局串联起来。看到 $d_p$ 时应立即利用 $ed_p-1$ 是 $p-1$ 倍数这一关系分解 $n$；得到明文后，还要检查响应头和隐藏路由补齐其余参数。生成 UUID 时需注意字段会截取低位，并区分“Python 3.14”与不存在的“Python 14”。
