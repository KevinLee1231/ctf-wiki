# week1简单套娃

## 题目简述

附件是一段五层嵌套编码，依次使用社会主义核心价值观编码、阴阳怪气编码、摩斯密码、W 型栅栏和凯撒移位。每一层解码后的字符特征会提示下一层的类型。

## 解题过程

按外层到内层依次还原：

1. 第一层只由“富强、民主、文明……”十二个词组成，这是社会主义核心价值观编码。其本质是先把 UTF-8 字节写成十六进制，再用十二个词表示十六进制数字；按该映射还原后得到大量“就 这 ¿”和“不 会 吧 ？”短语。
2. 阴阳怪气编码把“就 这 ¿”映射为 0、“不 会 吧 ？”映射为 1。每个字符前还有一个类型位：0 后读取 8 位 ASCII，1 后读取 16 位 Unicode。按此规则分帧，得到只含 `1`、`5` 和 `!` 的文本。
3. 该文本是替换过符号的摩斯密码：`1` 表示划、`5` 表示点、`!` 分隔字符。解码后得到：

```text
0W_1QIN4WN_D_FR{DL1C_JYLPDNA3GCY}X1S3J
```

4. 对上一步结果尝试标准 W 型栅栏解密。栏数为 5 时得到：

```text
0IRLXP{NW4DD1NLW_N1AS3C_1D_G3CJ_QFYYJ}
```

5. 最后一层是凯撒密码。字母向后移动 15 位后，前缀变为 `0XGAME{`；按照题目规定的大小写规范化前缀，最终得到：

```text
0xGame{CL4SS1CAL_C1PH3R_1S_V3RY_FUNNY}
```

## 方法总结

- 核心方法：根据每层输出的字符集识别下一种编码，并用已知 flag 前缀确定栅栏栏数和凯撒位移。
- 识别特征：固定的十二个价值观词、重复的两类短语、点划分隔结构、字符重排和统一字母偏移分别对应五层编码。
- 注意事项：阴阳怪气编码不是简单按 8 位分组，必须处理前导类型位；摩斯替换关系、栅栏是否保留非字母字符以及凯撒方向都要用中间结果验证。
