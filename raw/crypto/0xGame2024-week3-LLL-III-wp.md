# LLL-III

## 题目简述

题目使用已知参数的 LCG，但每轮只输出状态的高位：

$$
out_i=x_i\gg115
$$

其中 $x_{i+1}\equiv ax_i+b\pmod m$。已知 20 个连续高位，目标是利用每个状态未知低位均小于 $2^{115}$ 的约束，通过 LLL 恢复第一个完整状态，再反推 256 位 seed。

## 解题过程

令：

$$
l_i=out_i\cdot2^{115},\qquad x_i=l_i+r_i,\qquad0\le r_i<2^{115}
$$

代入 LCG 递推式：

$$
r_{i+1}\equiv ar_i+al_i+b-l_{i+1}\pmod m
$$

进一步写成：

$$
r_i\equiv A_i r_0+B_i\pmod m
$$

其中 $A_0=1,B_0=0$，并递推：

$$
A_i\equiv aA_{i-1}\pmod m
$$

$$
B_i\equiv aB_{i-1}+al_{i-1}+b-l_i\pmod m
$$

在 21 维格中放入 19 条模数对角向量，再放入 $(A_1,\ldots,A_{19},1,0)$ 与 $(B_1,\ldots,B_{19},0,2^{115})$。格中存在：

$$
(r_1,\ldots,r_{19},r_0,2^{115})
$$

这一所有坐标都约为 115 位的短向量，LLL 可以将其恢复。

完整 SageMath 脚本如下：

```python

low_bound = 2^115
known_high = [value * low_bound for value in out]

A = [1]
B = [0]
for i in range(1, len(known_high)):
    A.append((a * A[-1]) % m)
    B.append(
        (a * B[-1] + a * known_high[i - 1] + b - known_high[i]) % m
    )

lattice = Matrix(ZZ, 21, 21)
for i in range(19):
    lattice[i, i] = m

lattice[19] = A[1:] + [1, 0]
lattice[20] = B[1:] + [0, low_bound]

reduced = lattice.LLL()
short = next(
    row for row in reduced.rows()
    if abs(Integer(row[-1])) == low_bound
)

if short[-1] < 0:
    short = -short

r0 = Integer(short[-2])
assert 0 <= r0 < low_bound

x0 = known_high[0] + r0
seed = ((x0 - b) * inverse_mod(a, m)) % m
flag = "0xGame{" + md5(str(seed).encode()).hexdigest() + "}"

print(f"r0 = {r0}")
print(f"seed = {seed}")
print(flag)
```

## 方法总结

截断 LCG 的关键不是直接枚举 $2^{115}$ 个低位，而是把多个连续状态的模线性关系联立。所有未知低位都受到同一上界约束，因此可以被编码为一个异常短向量；样本越多，LLL 区分该向量与普通格向量就越容易。恢复首状态后，再逆一次初始化公式即可得到 seed。
