# IOT Project 2

## 题目简述

第二关仍围绕联网烤面包机。服务端提示 webhook 的两个字段分别包含普通文本和 hashtag；真正线索被发布到公开 Twitter 账号，账号简介中的十六进制字符串编码了 flag。

## 解题过程

查看站点源码和 `robots.txt`，可确认请求参数的语义：

```text
value1 = normal text
value2 = hashtag
```

使用精确短语搜索公开推文中的 `normal text`，并结合题目中的 toaster 主题筛选账号。找到对应烤面包机账号后，检查账号简介，而不只查看单条推文。

简介中的十六进制字符串为：

```text
554d444354462d7b7477336574216e675f346e645f7430407374316e677d
```

直接还原：

```python
bio = "554d444354462d7b7477336574216e675f346e645f7430407374316e677d"
print(bytes.fromhex(bio).decode())
```

得到：

```text
UMDCTF-{tw3et!ng_4nd_t0@st1ng}
```

## 方法总结

这题的 Web 服务负责暴露搜索锚点，公开社交账号负责承载结果。应把题面中的固定短语、参数含义和设备主题组合搜索，并检查账号级元数据。十六进制只是最后一层表示，不应把主要工作误判成密码学。
