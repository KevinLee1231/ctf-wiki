# CSEC Invasion 1

## 题目简述

页面表面只有外星人入侵主题，没有交互式漏洞。题目考查最基础的客户端信息泄露：flag 被开发者留在 HTML 注释中。

## 解题过程

使用浏览器的“查看网页源代码”，而不是只看渲染后的 DOM。在 HTML 注释中可以直接找到：

```html
<!-- UMDCTF-{@l13ns_@r3_b3tt3r_th@n_hum@ns} -->
```

因此 flag 为：

```text
UMDCTF-{@l13ns_@r3_b3tt3r_th@n_hum@ns}
```

## 方法总结

HTML 注释不会显示在页面正文，却仍完整发送给客户端。Web 题初查应包括响应源码、静态资源、脚本和常见元数据文件；不要把“页面看不见”误认为“用户拿不到”。
