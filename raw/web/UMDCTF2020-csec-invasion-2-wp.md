# CSEC Invasion 2

## 题目简述

题面从外星人转向“robots”，直接提示网站根目录的 `robots.txt`。该文件本来用于向爬虫声明抓取规则，但任何访问者都能读取。

## 解题过程

访问：

```text
/robots.txt
```

响应中直接包含：

```text
UMDCTF-{d0m0_@r1g@t0_mr_r0b0t0}
```

`robots.txt` 不是权限控制机制；即使其中写了 `Disallow`，也只是请求守规矩的爬虫不要访问相应路径。

## 方法总结

题目关键词常指向标准 Web 元数据。检查 `robots.txt` 的价值不仅是寻找直接 flag，也包括发现未在页面链接中出现的后台或敏感路径；真正的保护仍必须由服务端鉴权实现。
