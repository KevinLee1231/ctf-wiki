# Invest on Matrix

## 题目简述

题目没有附件和靶机。官网题干说明这是 “Invest in hints 3.0” 版本：现在有一个 `25 * 25` 的矩阵，hint 会把该矩阵按从左到右、从上到下的顺序划分为若干 `5 * 5` 子矩阵，并依次给出每个子矩阵按行优先扁平化后的值。例如 `1 pts` 对应 `matrix[0:5, 0:5]`，`2 pts` 对应 `matrix[0:5, 5:10]`，`6 pts` 对应 `matrix[5:10, 0:5]`。解得的预期输入需要用 `hgame{}` 包裹提交。

题目的关键机制是“二维码图像被矩阵块打散”，不是常规图片隐写。WP 中需要保留的是分块尺寸、黑白值扁平化方式和还原二维码的方向。

## 解题过程

拿到 hint 后，把每组 25 个黑白值按 `5 * 5` 还原成小块，再按题目给出的顺序拼接回完整二维码。由于编号已经对应块位置，`idx = pts - 1`，块坐标可以直接按 `x = idx % 5`、`y = idx // 5` 放置。

![还原出的二维码](HGAME2026-invest-on-matrix-wp/还原出的二维码.png)

实现上可以把每个小块转成二值图像，然后按索引写回大图：

```python
from PIL import Image

BLOCK = 5

def draw_qr(chunks, width_blocks, out_path):
    img = Image.new("1", (width_blocks * BLOCK, width_blocks * BLOCK), 1)
    for idx, values in enumerate(chunks):
        bx = idx % width_blocks
        by = idx // width_blocks
        for i, v in enumerate(values):
            x = bx * BLOCK + (i % BLOCK)
            y = by * BLOCK + (i // BLOCK)
            img.putpixel((x, y), 0 if v else 1)
    img.save(out_path)
```

二维码修复后扫码即可进入最终验证步骤。若局部块缺失，可以借助二维码定位点、格式信息和纠错能力补齐。

## 方法总结

- 核心技巧：把扁平化的黑白矩阵按固定块大小还原成二维码。
- 识别信号：题面出现矩阵投资、hint 购买、固定长度黑白数组时，应考虑图像块重组。
- 复用要点：二维码题先恢复定位点和整体尺寸，缺块不一定致命，二维码自带纠错可以容忍少量错误。
