# week4Couple Icon

## 题目简述

附件包含两层加密压缩包和一张使用 WBStego 隐写的 BMP。第一层密码由提示字符串 `lxx` 的小写 MD5 得到，第二层密码是题面所指生日 `000507`，最后使用 `lxx` 作为隐写密钥提取 flag。

## 解题过程

`hint.txt` 指出第一层压缩包密码由 32 个小写字符组成。结合附件中唯一明确的候选字符串 `lxx`，计算其 MD5 十六进制摘要：

```text
md5("lxx") = 73fe8f327b36932eb9d58b02ca722062
```

使用该值解开第一层，得到 `Pretender.zip`。题面提示下一层密码是生日，因此按六位日期格式生成候选值进行字典测试，而不是无约束爆破所有字符。正确密码为：

```text
000507
```

解压后得到 `Pretender.bmp`。BMP 图像本身可以正常显示，但其中包含 WBStego 写入的数据。使用 WBStego 4 Open 打开该文件，选择提取隐藏数据，并输入密钥 `lxx`，即可恢复 flag：

```text
0xGame{si1ence_virgin_l0ve}
```

## 方法总结

本题的关键是让每个提示约束对应一层：`32 位小写` 指向 MD5 的十六进制形式，`生日` 把密码空间缩小到日期，最终的 BMP 与重复出现的 `lxx` 则指向带密钥的 WBStego 提取。这样可以避免无依据地枚举大量压缩与隐写工具。
