# 赛博天尊

## 题目简述

程序取 `0xGame{...}` 内部内容，按连字符 `-` 分成五段并解析为整数，然后检查五个线性方程。无需爆破，只要从反编译结果准确抄出系数和常数，用符号求解器解线性方程组即可。

## 解题过程

在 IDA 中沿输入解析和成功分支回溯，可以看到五个变量及对应约束。将反编译表达式原样转换为 SymPy 方程；第二个判断在源码中表现为等式两侧都含变量，保留原结构交给求解器即可：

```python
from sympy import Eq, solve, symbols

v7, v8, v9, v10, v11 = symbols("v7 v8 v9 v10 v11")

equations = [
    Eq(7*v9 + 5*(v8+v11) + 2*(v10+4*v7), 0x12021DE669FC2),
    Eq(v9+v10+2*v10+2*(v11+v7)+v8,
       0x159BFFC17D045 - 2*(v9+v10+2*v10+2*(v11+v7))),
    Eq(v10+v9+v11+2*v9+2*(v9+v11+2*v9)+2*(v8+4*v7),
       0xACE320D12501),
    Eq(v8 + 2*(v7+v11+v9+2*v10), 0x733FFEB3A4FA),
    Eq(v8 + 7*v11 + 8*(v9+v10) + 5*v7, 0x1935EBA54EB28),
]

solution = solve(equations, (v7, v8, v9, v10, v11), dict=True)[0]
parts = [solution[x] for x in (v7, v8, v9, v10, v11)]
flag = "0xGame{" + "-".join(f"{int(x):x}" for x in parts) + "}"
print(flag)
```

程序以十六进制片段构造最终输入，因此脚本把每个整数格式化为不带 `0x` 前缀的小写十六进制，并按原顺序用 `-` 连接。

## 方法总结

这类题的难点通常不是求解，而是准确恢复解析格式、变量顺序、符号和常数。应先把每条比较规范化为方程，再用解出的值回填原程序验证；不要把连字符分隔、十六进制格式等输入层约束遗漏在数学求解之外。
