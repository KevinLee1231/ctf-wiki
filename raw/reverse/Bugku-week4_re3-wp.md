# week4_re3

## 题目简述

题目文件名是 `re3.exe`，但实际不是 PE 可执行文件，而是纯文本。文件内容直接包含 flag，因此不需要 Ghidra 或动态调试。

## 解题过程

直接读取文件内容：

```powershell
Get-Content -Raw -LiteralPath "re3.exe"
```

得到：

```text
flag{ff6cf093-f357-428d-a642-16b10702bae7}
```

## 方法总结

- 核心技巧：先识别文件真实格式，不要被扩展名误导。
- 识别信号：`.exe` 文件体积异常小，`file`/十六进制查看没有 `MZ` 头时，应先按文本或数据文件处理。
- 复用要点：逆向题第一步仍是文件类型确认；不是所有 `.exe` 都需要反编译。
