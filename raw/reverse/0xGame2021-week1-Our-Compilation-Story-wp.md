# week1Our Compilation Story

## 题目简述

附件中的 Python 逻辑先反转整数数组，再把每个元素与其后第 3 个元素异或。异或结果按字符解释后就是完整 flag。

## 解题过程

编码数组共有 37 个元素。反转后仅遍历到 `len(arr)-3`，因此输出 34 个字符；每个字符满足：

$$
plain_i=arr_i\oplus arr_{i+3}
$$

直接按原逻辑执行：

```python
arr = [
    21, 44, 45, 104, 31, 30, 26, 121, 65, 125, 23, 112, 77, 46, 47,
    126, 89, 112, 7, 109, 7, 88, 10, 105, 104, 59, 54, 91, 83, 98, 32,
    54, 15, 65, 113, 119, 113,
]

arr.reverse()
flag = "".join(chr(arr[i] ^ arr[i + 3]) for i in range(len(arr) - 3))
print(flag)
```

输出为：

```text
0xGame{Th3_10ng_w4y_w3_901ng_fr33}
```

## 方法总结

本题只需忠实还原数组变换的执行顺序：先整体反转，再做固定距离为 3 的异或。处理此类短脚本题时，应特别核对遍历边界和变换先后，避免把原数组与反转后数组混用。
