# Twilight Zone

## 题目简述

附件是 .NET 程序。程序先用日期条件阻止正常进入，再校验一段固定输入，并以其 MD5 派生 AES 密钥解密内置密文。通过反编译可以分别绕过时间门槛、恢复口令和还原明文。

## 解题过程

在 dnSpy 或 ILSpy 中定位按钮事件。日期比较只控制是否继续，可以修改系统日期、修补条件跳转，或直接调用后续解密函数。校验逻辑对应的正确输入为：

```text
mLxCKPYXjSMUcEKu
```

程序计算该字符串的 MD5，并按源码中重叠取字节的方式拼出 AES 密钥；随后用 AES-ECB 解密内置密文并去除 PKCS#7 填充。复现时要逐字节照抄密钥拼接顺序，不能把 MD5 十六进制文本当作原始摘要。

解密结果为：

```text
UMDCTF-{w3lcome_t0_the_tw1light_z0ne}
```

## 方法总结

.NET 程序的日期门槛属于控制流障碍，不是密码学核心。先在反编译器中把事件处理链拆开，再区分字符串、MD5 原始字节和 AES 密钥的具体表示，才能避免因编码层次错误得到无意义明文。
