# week3Installer

## 题目简述

附件核心是 Python 字节码 `py.pyc`。反编译后可见程序把 flag 转成整数数组，连续 16 轮加上一个“每轮翻倍并反转”的密钥数组；`output.txt` 保存最终整数。记录每轮实际加数并逆序减回即可恢复 flag。

## 解题过程

先反编译 PYC：

```bash
uncompyle6 py.pyc > recovered.py
```

加密每轮执行 `arr[j] += key[j]`，随后把所有 key 翻倍并整体反转。下面的代码保留初始 key 和 `output.txt` 的完整结果，先生成 16 轮使用过的 key 状态，再按相反顺序减回：

```python
key = [
    238, 257, 150, 137, 167, 169, 184, 193, 210, 147, 219, 128, 140,
    135, 185, 242, 204, 128, 132, 159, 222, 173, 226, 159, 207, 169,
    154, 156, 216, 139, 168, 187, 220, 237, 207, 187, 218, 138, 218,
    178, 246, 239, 246, 241,
]

encrypted = [
    15728448, 16362025, 13718731, 13740602, 11425044, 13216326,
    10048823, 13740603, 12757531, 12255100, 15138636, 12408061,
    11228430, 10289095, 10114289, 14723575, 11272070, 9524519,
    10267251, 12517282, 11796345, 13653174, 12495389, 13172636,
    11468724, 9458930, 8956506, 12320680, 15291551, 11119205,
    9568155, 10201663, 10398270, 14745427, 10944395, 13260012,
    13194479, 11053619, 12145871, 11184688, 11359448, 11774503,
    16602251, 15662990,
]

round_keys = []
current = key[:]

for _ in range(16):
    round_keys.append(current[:])
    current = [value * 2 for value in current][::-1]

plain = encrypted[:]
for round_key in reversed(round_keys):
    plain = [value - delta for value, delta in zip(plain, round_key)]

print("".join(chr(value) for value in plain))
```

输出为：

```text
0xGame{b373edd6-2cea-11ec-8d3d-0242ac130003}
```

## 方法总结

本题链路为“反编译 PYC → 识别逐轮可逆变换 → 重建每轮 key → 逆序执行减法”。`uncompyle6 -o` 的参数是输出目录，不是输出文件；直接重定向到 `recovered.py` 更明确。即使反编译器失效，也可从字节码中确认同样的加法、翻倍和反转顺序。
