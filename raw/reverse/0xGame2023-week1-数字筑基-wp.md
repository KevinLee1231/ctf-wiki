# 数字筑基

## 题目简述

程序给出一串二进制表示的数据，要求输入其十六进制形式。这里的“输入十六进制”指输入由十六进制字符组成的 ASCII 字符串，而不是把转换后的字节直接发送给程序。

## 解题过程

将题目输出按 8 位一组恢复原始字节，再把这些字节表示为十六进制文本。本题的二进制数据先还原为 ASCII 字节串 `0xCAFEBABE`，因此最终应输入的是该字节串的十六进制编码：

```python
bits = "..."  # 程序给出的二进制字符串
raw = int(bits, 2).to_bytes((len(bits) + 7) // 8, "big")
answer = raw.hex()
assert raw == b"0xCAFEBABE"
assert answer == "30784341464542414245"
print(answer)
```

把输出的十六进制字符原样输入程序即可。例如原始字节为 `0x30` 时，应键入两个字符 `30`，而不是键入字符 `0`。

## 方法总结

解题时要区分“数值”“字节”和“字节的文本表示”。CyberChef 外链只是便捷实现，关键步骤是先从二进制位恢复字节，再把字节编码为十六进制 ASCII；正文中的本地脚本可在不访问外链时完整复现。
