# week1Packet

## 题目简述

程序经过 UPX 压缩，脱壳后可见一个逐字符校验函数。该函数把 `(a & ~b) | (b & ~a)` 写成较绕的形式，但它逐位等价于 `a ^ b`，因此输入字符可由两组常量直接异或恢复。

## 解题过程

从节名或 UPX 标记确认壳后，先保留原文件并输出到新文件：

```bash
upx -d Packet.exe -o Packet.unpacked.exe
```

脱壳后的核心函数为：

```c
int encrypt(int a, int b)
{
    return (a & ~b) | (b & ~a);
}
```

对每一位而言，只有 `a`、`b` 不同时结果为 1，所以该表达式就是异或。校验条件 `encrypt(input[i], check[i]) == enc[i]` 可逆为 `input[i] = check[i] ^ enc[i]`：

```python
check = [
    161, 15, 188, 111, 218, 169, 159, 94, 41, 246, 197, 228, 110, 242,
    177, 56, 27, 1, 17, 256, 256, 50, 233, 65, 104, 2, 4, 6, 42, 112,
    55, 107, 48, 93, 130, 232, 37, 87, 242, 130,
]

enc = [
    145, 119, 251, 14, 183, 204, 228, 56, 17, 148, 253, 133, 92, 145,
    132, 92, 125, 103, 39, 308, 309, 10, 216, 35, 13, 48, 101, 62, 19,
    69, 84, 82, 81, 62, 176, 217, 19, 51, 195, 255,
]

flag = "".join(chr(left ^ right) for left, right in zip(check, enc))
print(flag)
```

输出为：

```text
0xGame{f8b8a2c5dff64581be2a895c9ac216d1}
```

## 方法总结

本题链路为“识别并脱 UPX → 化简位运算 → 逆向逐字节校验”。UPX 项目链接不是解题必要条件；WP 已保留识别对象、脱壳命令、布尔等价关系和完整恢复代码。
