# Wake Me Up Before I Go Go

## 题目简述

题目是 Go 编译的二进制。主函数开头有一个恒真条件导致提前返回，真正的 flag 计算代码位于不可达分支中：它计算一个大整数乘积，再把结果的字节表示转成字符串。

## 解题过程

Go 二进制较大，但可以从 `main.main` 和字符串交叉引用定位用户逻辑。跳过早期返回后，核心等价于：

```go
c, _ := new(big.Int).SetString(
    "13402705915692712146309925734755389833778259",
    10,
)
c.Mul(c, big.NewInt(5*13*43*13))
fmt.Println(string(c.Bytes()))
```

无需修补整个程序，直接用 Python 重现同一大端整数转字节语义：

```python
c = 13402705915692712146309925734755389833778259
c *= 5 * 13 * 43 * 13
plain = c.to_bytes((c.bit_length() + 7) // 8, "big")
print(plain.decode())
```

计算结果为：

```text
UMDCTF-{1_l0v3_g000}
```

## 方法总结

编译器保留的不可达代码仍可能包含完整解题逻辑。Go 的 `big.Int.Bytes()` 返回无符号大端绝对值，因此 Python 必须使用大端、最小长度转换。题目中的恒真返回只是控制流障碍，不需要绕过后再运行未知程序。
