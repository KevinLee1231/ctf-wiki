# week1Neverland

## 题目简述

程序用递归数列 $a_0=7$、$a_1=8$、$a_n=3a_{n-1}+4a_{n-2}$ 生成密钥流，再与 40 个整数异或输出 flag。部分索引极大，直接递归无法完成；关键是利用闭式和 32 位无符号整数溢出。

## 解题过程

特征方程为 $r^2-3r-4=0$，根为 $4$ 和 $-1$。结合初值得到：

$$
a_n=3\times4^n+4\times(-1)^n
$$

源码返回类型是 `unsigned int`，所以每次运算实际都在模 $2^{32}$ 下进行。原 WP 写成模 `0x10000000` 并不正确。对于 $n\ge16$，$4^n=2^{2n}$ 已含有至少 32 个因子 2，因此第一项在模 $2^{32}$ 下为 0，数列只由奇偶性决定：偶数索引为 `4`，奇数索引为 $2^{32}-4=4294967292$。

下面保留源码中的完整数组并直接按闭式求值：

```python
enc = [
    786380, 3221225348, 50331715, 221, 89, 97, 12582791, 805306421,
    49049, 3145778, 54, 4294967241, 4294967192, 48, 4294967199,
    4294967244, 4294967240, 98, 4294967193, 48, 4294967240,
    4294967194, 4294967237, 4294967198, 50, 4294967236, 4294967240,
    96, 61, 53, 61, 4294967243, 52, 60, 4294967199, 101, 101, 51, 102, 121,
]

idx = [
    9, 15, 12, 3, 2, 16, 11, 14, 7, 10, 46, 45, 43, 46, 47, 45, 47,
    40, 49, 58, 49, 51, 51, 43, 50, 55, 55, 56, 60, 48, 4294967246,
    4294967187, 4294967256, 4294967282, 4294967263, 4294967152,
    4294967154, 4294967248, 4294967206, 4294967194,
]

modulus = 1 << 32

def sequence(index):
    sign = 1 if index % 2 == 0 else -1
    return (3 * pow(4, index, modulus) + 4 * sign) % modulus

flag = "".join(
    chr((sequence(index) ^ value) & 0xff)
    for index, value in zip(idx, enc)
)
print(flag)
```

输出为：

```text
0xGame{1e625d4c04fe44f9b684d919708caa7b}
```

## 方法总结

本题的核心不是“把递归改成循环”这么简单，而是识别线性递推的闭式并保留 C/C++ 的 32 位溢出语义。外部文章中的斐波那契优化和通项推导已被正文中的特征方程、闭式与模运算完整替代，因此无需依赖外链。
