# 代码启示录

## 题目简述

题目附件是一个 Java JAR。程序没有对 flag 做额外编码或加密，关键在于从编译后的类中恢复 `Main` 类的校验逻辑。

## 解题过程

使用 JADX 打开 JAR，定位程序入口 `Main`。反编译结果显示，程序直接将用户输入与一个写死的字符串比较，因此无需继续追踪其他类或运行时状态。关键判断为：

```java
if (userInput.equals("0xGame{038e8685-0a59-a868-ed44-979261e99faf}")) {
    System.out.println("恭喜你，打开了神秘的门！");
}
```

其中的比较常量即为：

```text
0xGame{038e8685-0a59-a868-ed44-979261e99faf}
```

## 方法总结

对于无混淆的 Java JAR，先从入口类、输入读取点和字符串比较点入手。该题的 flag 作为明文常量参与比较，反编译 `Main` 类即可完整恢复。
