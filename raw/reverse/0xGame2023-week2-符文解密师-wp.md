# 符文解密师

## 题目简述

程序把用户输入与硬编码字符串 `deadc0de` 直接比较，比较成功后进入输出 flag 的分支。题目没有额外加密，主要考查定位入口函数和识别字符串比较。

## 解题过程

用 IDA 打开程序，定位 `main` 并按 `F5` 反编译。沿输入函数向下查看，可将关键判断整理为：

```c
if (strcmp(input, "deadc0de") == 0) {
    print_flag();
}
```

运行程序并输入：

```text
deadc0de
```

比较成立后程序输出：

```text
0xGame{18f03f86-9783-62b5-466d-fc84c28bad3b}
```

也可以从字符串窗口搜索 `deadc0de`，查看交叉引用并确认它确实参与成功路径判断，从而排除诱饵文本。

## 方法总结

基础校验题优先检查输入函数、比较函数和可疑常量。反编译器给出的伪代码应结合交叉引用与控制流核实；确认是直接比较后，不需要再引入不相关的解密步骤。
