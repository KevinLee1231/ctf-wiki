# week4re1

## 题目简述

题目给出一个自定义虚拟机程序。程序把输入的 42 个字符逐字节变换，再与内置数组比较。目标是还原虚拟机指令语义，并逆运算得到正确输入。

## 解题过程

分析解释器的分发逻辑，可以整理出 8 个有效操作码：

| 操作码 | 语义 |
| --- | --- |
| `1` | `R[a] ^= R[b]` |
| `2` | `R[a] += R[b]` |
| `3` | `R[a] -= R[b]` |
| `4` | `R[a] <<= R[b]` |
| `5` | `R[a] >>= R[b]` |
| `6` | `mem[a] = R[b]` |
| `7` | `R[a] = mem[b]` |
| `8` | `R[a] = b` |

将字节码按每条指令三个整数反汇编后，可以看到 42 组结构完全相同。第 $i$ 组先取输入 `mem[i]`，与随轮次递增的 `R2` 异或，再左移 $i \bmod 3$ 位，最后加上递增后的 `R2`，并写入 `mem[100+i]`。其等价关系为：

$$
c_i = \left(\left(m_i \oplus (32+i)\right) \ll (i \bmod 3)\right) + (33+i)
$$

这里的加法发生在移位完成后，因此逆运算为：

$$
m_i = \left((c_i-33-i) \gg (i \bmod 3)\right) \oplus (32+i)
$$

把程序内置的 42 个比较值代入即可直接恢复输入：

~~~python
enc = [
    0x67, 0xBC, 0x12F, 0x68, 0xE3, 0x136,
    0x67, 0x50, 0x161, 0x77, 0x5B, 0xA8,
    0x41, 0x2E, 0x97, 0x4E, 0xD5, 0x46,
    0x52, 0x42, 0x65, 0x43, 0x3B, 0xA0,
    0x3A, 0xF8, 0x6B, 0x95, 0x5F, 0x66,
    0x49, 0x50, 0x221, 0xB8, 0x137, 0xC8,
    0x66, 0x140, 0xD3, 0xB8, 0x141, 0x11A,
]

flag = "".join(
    chr(((value - 33 - i) >> (i % 3)) ^ (32 + i))
    for i, value in enumerate(enc)
)
print(flag)
~~~

运行结果为：

~~~text
flag{af3fd248-41b4-4884-9f6b-747878be8e74}
~~~

## 方法总结

这类 VM 题的重点是先建立“操作码—语义”映射，再观察反汇编结果中的重复模板。确认每轮的寄存器状态后，可直接把整段字节码归纳为逐字节公式，不必保留数百行同构指令。逆序执行减法、右移和异或即可还原输入。
