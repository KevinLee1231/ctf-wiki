# week1re1

## 题目简述

附件是一个 Windows 可执行文件。程序没有对 flag 做实质性加密，正确字符串以明文形式保存在二进制中。

## 解题过程

用 IDA 打开 `re1.exe`，先查看 Strings 窗口并搜索 `0xGame{`。字符串可直接定位到：

```text
0xGame{be9d9fee-7d45-4a3e-a105-802b3221665d}
```

继续查看该字符串的交叉引用，可以确认它由程序的输出逻辑直接使用，不是无关的诱饵字符串，因此提交该值即可。

## 方法总结

- 核心方法：静态查看字符串并用交叉引用确认其用途。
- 识别特征：程序体积小，题目没有复杂输入校验，二进制中能检索到完整 flag 格式。
- 注意事项：仅看到疑似 flag 时仍应检查引用位置，避免误交未被执行路径使用的诱饵字符串。
