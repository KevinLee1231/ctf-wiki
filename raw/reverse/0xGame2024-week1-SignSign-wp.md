# SignSign

## 题目简述

附件是一个 Windows 可执行文件。程序运行时只输出 flag 的后半段，但完整 flag 的两部分都以明文字符串保存在程序中，因此本题的关键是查看二进制中的静态字符串，而不是只依赖运行结果。

## 解题过程

用 IDA 打开 `SignIn.exe`。主函数的反汇编结果中没有直接出现完整 flag，说明还需要检查程序保存的其他静态数据。

按 `Shift+F12` 打开 Strings 窗口，可以找到两段互补的字符串：

```text
0xGame{S1gn1n_h3r3_4nd
_b3g1n_Reversing_n0w}
```

按照出现顺序拼接两段内容，得到完整 flag：

```text
0xGame{S1gn1n_h3r3_4nd_b3g1n_Reversing_n0w}
```

## 方法总结

程序实际输出的内容不一定等于二进制中保存的全部信息。遇到签到型逆向题时，可以先运行程序观察行为，再检查 Strings 窗口、只读数据段及字符串交叉引用；本题直接拼接两段明文字符串即可还原 flag。
