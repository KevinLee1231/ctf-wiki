# week1re4

## 题目简述

附件使用 TEA 对 flag 分组加密。程序中可以提取 128 位密钥和密文；每个分组由两个 32 位无符号整数组成，执行 32 轮运算，轮常量为 `0x9e3779b9`。

## 解题过程

在反编译结果中寻找固定轮数 32、移位量 4 和 5，以及常量 `0x9e3779b9`，即可判断为标准 TEA。TEA 的一个分组是 64 位，密钥是四个 `uint32_t`；解密时从 `delta * 32` 开始，按加密的逆序先还原 `v1`、再还原 `v0`，并依赖 `uint32_t` 的自然溢出完成模 $2^{32}$ 运算。

从程序中按原始字节序提取密钥和密文，每两个 32 位整数作为一个分组解密：

~~~c
#include <stdint.h>
#include <string.h>
#include <stdio.h>
#include <stdlib.h>

void encrypt(uint32_t *v, uint32_t *k)
{
    uint32_t v0 = v[0], v1 = v[1], sum = 0, i;
    uint32_t delta = 0x9e3779b9;
    uint32_t k0 = k[0], k1 = k[1], k2 = k[2], k3 = k[3];
    for (i = 0; i < 32; i++) {
        sum += delta;
        v0 += ((v1 << 4) + k0) ^ (v1 + sum) ^ ((v1 >> 5) + k1);
        v1 += ((v0 << 4) + k2) ^ (v0 + sum) ^ ((v0 >> 5) + k3);
    }
    v[0] = v0;
    v[1] = v1;
}

void decrypt(uint32_t *v, uint32_t *k)
{
    uint32_t v0 = v[0], v1 = v[1], sum = 0x9e3779b9 * 32, i;
    uint32_t delta = 0x9e3779b9;
    uint32_t k0 = k[0], k1 = k[1], k2 = k[2], k3 = k[3];
    for (i = 0; i < 32; i++) {
        v1 -= ((v0 << 4) + k2) ^ (v0 + sum) ^ ((v0 >> 5) + k3);
        v0 -= ((v1 << 4) + k0) ^ (v1 + sum) ^ ((v1 >> 5) + k1);
        sum -= delta;
    }
    v[0] = v0;
    v[1] = v1;
}

int main()
{
    uint32_t key[4] = {0x795F6F64, 0x6B5F756F, 0x5F776F6E, 0x3F616574};
    uint32_t enc[] = {3293258237, 3797453781, 2711996313, 3260442805,
                      1056803237, 1218477302, 896926073, 1670822367,
                      2779477777, 2262174553, 226803321, 4259941008, 0};
    for (int i = 0; i < 12; i += 2) {
        decrypt(enc + i, (uint32_t *)key);
    }
    printf("%s", (char *)(enc));
    return 0;
}
~~~

## 方法总结

- 核心方法：通过 TEA 的轮常量和移位结构识别算法，提取四个密钥字和密文分组后执行逆运算。
- 识别特征：常见组合是 32 轮、`0x9e3779b9`、左右移 4/5 位，以及两个 32 位状态交替更新。
- 注意事项：必须确认 TEA、XTEA 等变体的更新顺序，并保持 32 位无符号溢出；密钥和密文的端序错误也会使结果完全不可读。
