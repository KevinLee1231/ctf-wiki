# Question Bash

## 题目简述

远程环境提供受限的 shell 命令执行，目标文件名和内容都不可直接枚举。关键点是 shell 会在命令执行前展开通配符，因此可以用长度模式匹配未知文件名。

## 解题过程

题目目标文件名恰好是 8 个字符。问号通配符每个只匹配一个字符，所以可以用八个问号代替未知名称：

```sh
cat ????????
```

若当前目录还包含其他同长度对象，可先用同一模式观察展开结果，或给模式补充已知前后缀。服务中的目标命中后直接打印：

```text
UMDCTF-{s0me_questions_h4ve_answ3rs}
```

仓库 README 中记录的哈希与实际公开构建不一致；本题采用部署文件与输出中一致的明文作为依据。

## 方法总结

受限 shell 中不仅要检查可调用的命令，还要考虑 shell 自身在执行前完成的 glob 展开。`?` 表示单个任意字符，特别适合在已知文件名长度时绕过不能直接输入名称的限制。
